# TaskFlow

TaskFlow is a modern responsive task management landing page built using HTML, CSS, and JavaScript.

## Features
- Responsive Navbar
- Hero Section
- Features Section
- Pricing Plans
- Contact Form Validation
- Dark Mode
- Dynamic Plan Selection
- Mobile Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Netlify
